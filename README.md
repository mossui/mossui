# Moss UI

## Colors

### NPM

```bash
# npm
npm i @mossui/colors
```

```css
/* main.css */
@import '@mossui/colors';
```

### CDN

```html
<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/@mossui/colors/dist/colors.min.css"
/>
```

### Usage

```html
<h1>Hello World</h1>
```

```css
h1 {
  color: var(--color-red-500)
}
```
